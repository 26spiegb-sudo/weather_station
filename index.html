Weather Station Project

Engineering Design Portfolio

Description

This project is an Arduino-based weather station that measures and displays temperature, humidity, barometric pressure, and maximum/minimum temperature in real time. The device uses a BME280 sensor and outputs readings to a TFT display on an Arduino Mega. Components are housed in a custom 3D-printed enclosure.

Inspiration

Arduino Easy Weather Station With BME280 Sensor by Nick Koumaris on Instructables.

Source: Instructables Link

How This Project Differs

The original tutorial uses an Arduino Mega with a 16x2 LCD shield. This project upgrades to a TFT color shield with display cycling, Fahrenheit readings, max/min temperature tracking, and a fully enclosed 3D-printed case.

Design Considerations

Focus was on readability, accurate sensor readings, compact design, and ease of assembly.
